# Changelog

## [0.2.0] - 2026-04-06

- **Nova funcionalidade**: Suporte completo ao `exiftool` para leitura de metadados avançados
- **Nova funcionalidade**: Estrutura de categorias e sub-categorias inteligentes
- **Nova funcionalidade**: Barra de busca em tempo real para filtrar tags e valores
- **Nova funcionalidade**: Botões para expandir/recolher toda a árvore de metadados
- **Nova funcionalidade**: Exportação de metadados para JSON ou CSV
- **Nova funcionalidade**: Menu de contexto para copiar valores individuais
- **Melhoria**: Interface reescrita com TreeStore para melhor organização hierárquica
- **Melhoria**: Suporte a MakerNotes, XMP, IPTC, Composite, ICC Profile
- **Melhoria**: Tratamento de valores longos com truncamento automático
